# I know you

### Installation:
Note: This requires a recent version of NodeJS to work.
```shell
git clone git@github.com:maxnachlinger/react-websocket-example.git

cd react-websocket-example
cd client
npm i

cd ../server
npm i
```

### Running this app:
```shell
# start a webpack dev watch build
cd client
npm start

# start the server
cd ../server
npm start
```
The app runs at [http://your local ip address:3000]

### Client tests:
```shell
cd client
npm test
```
